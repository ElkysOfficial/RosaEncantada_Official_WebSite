# Palavras-chave

#seo/keywords

## Termos primários (alto volume, alta intenção)

- trufas artesanais
- bombons artesanais
- chocolate belga artesanal
- trufas para presente
- caixa de trufas
- doces para encomenda

## Termos de cauda longa (baixa concorrência, alta conversão)

- trufas para casamento [cidade]
- lembrancinhas de trufas para casamento
- caixa de trufas para presente
- trufa de pistache artesanal
- trufa de morango artesanal
- doces personalizados para evento
- mesa de doces para casamento [cidade]
- trufas sob encomenda
- bombom premium para presentear

## Por seção

| Seção | Keyword principal | Keywords secundárias |
|-------|------------------|--------------------|
| Hero | trufas artesanais | chocolate belga, doces artesanais |
| Sobre | confeitaria artesanal | trufas autorais, doces feitos a mão |
| Produtos | catálogo de trufas | sabores, bombons, caixas presente |
| Encomendas | trufas para casamento | lembrancinhas, doces para evento |
| Processo | como são feitas as trufas | trufas artesanais processo |
| Contato | encomendar trufas | falar com confeiteira |

## Termos a evitar

- "trufa de chocolate" (genérico, baixa intenção)
- "doceira" (concorre com termos genéricos)
- "artesanato" (confunde com não-comestível)
- "feito em casa" (impressão amadora)
