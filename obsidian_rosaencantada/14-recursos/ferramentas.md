# Ferramentas locais

#recursos/ferramentas

O que é usado para trabalhar no projeto.

## Editor

- **VS Code** — editor principal
- **Obsidian** — vault (este aqui)

## Plugins de Obsidian recomendados (cada um instala localmente)

> O `.obsidian/` está no `.gitignore` — escolha os seus. Sugestões úteis para este vault:

- **Templater** — automação de templates de notas novas
- **Dataview** — queries por tags em notas (ex.: "todas as notas com `#tarefa/backlog`")
- **Git** — sync do vault para o repo
- **Auto Link Title** — título de páginas web ao colar URL

## Navegadores para teste

- Chrome (Lighthouse)
- Edge (BrowserStack se precisar)
- Firefox (visual)
- Safari (mobile, ideal testar no iPhone real)

## CLI / shell

- **PowerShell 7+** — para os scripts `.ps1`
- **Git** — versionamento
- **FileZilla** — FTP para Hostinger (alternativa ao Gerenciador de Arquivos)

## Conversão de imagens

- Local (sem instalação): script `convert-images.ps1` usa System.Drawing nativo
- Inkscape (opcional): conversão SVG mais fiel
- Squoosh (online): otimização final
