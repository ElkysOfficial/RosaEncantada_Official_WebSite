## 📐 Como usar este vault

1. **Abra a pasta `obsidian/` como vault** no Obsidian (`Open folder as vault`).
2. Comece sempre por este README — ele é o ponto de partida.
3. Use [[backlinks]] e tags `#area/subarea` para conectar ideias.
4. Notas seguem o padrão **um arquivo, um conceito**.
5. Decisões importantes viram ADRs em `11-decisoes/` — uma vez registradas, são imutáveis.

## 🛡️ Convenções

- **Nomes de arquivo**: `kebab-case.md`, sem acentos.
- **Títulos H1**: humanizados, com acento.
- **Tags**: `#area/subarea` (ex.: `#seo/keywords`, `#produto/trufa`).
- **Datas**: ISO `2026-05-01`.
- **Links internos**: `[[wiki-link]]` sempre que possível.
- **Imagens**: pasta dedicada por tópico, prefixo do nome igual ao da nota.
